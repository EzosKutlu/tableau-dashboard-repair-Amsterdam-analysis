# tableau-dashboard-repair-Amsterdam-analysis
Interactive Tableau dashboard published on Tableau Public.

Live Dashboard (Tableau Public):
https://public.tableau.com/app/profile/ezgi.kutluer/viz/AmsterdamAirbnbdatareview/Dashboard1

About the Dashboard
This dashboard is based on Airbnb data for Amsterdam.
On the first page, the dashboard shows how total accommodation revenue and the number of guests change over time.
The second page focuses on trends between 2019 and 2024, highlighting changes over time. This page also includes visualizations that show how accommodation types and average prices by accommodation type vary across neighborhoods and years.

Tools Used
- Tableau Desktop
- Tableau Public
- Excel (data source)
- Tableau Extract (.hyper)

This dashboard is published on Tableau Public for portfolio purposes.
Data is read-only and used for demonstration.
